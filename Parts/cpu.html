Trouble Shooting Guide

Problem 1: System won't boot

Ensure CPU is properly seated in the socket.

Check for bent pins or debris.

Verify 8‑pin CPU power connector is plugged in.

Test with minimal hardware (one RAM stick, no GPU).

Problem 2: Overheating

Confirm cooler is mounted tightly.

Reapply thermal paste.

Clean dust from heatsink/fans.

Monitor temps in BIOS or HWMonitor.

Problem 3: Frequent crashes/blue screens

Disable overclocking in BIOS.

Run stress test (Prime95).

Check temps during load.

Swap CPU into another system if possible.
